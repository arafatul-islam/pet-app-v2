
# Pet Foster Center 

This is a pet foster app. User can give their pets for fostering for a while or as long as they want.
 


## Authors

- [@arafatul-islam](https://www.github.com/arafatul-islam)


## Features

- Light/dark mode toggle
- fully responsive
- pet foster option
- pet adoption option
- chat with pet owner
- pet shop 
- vet finder
- video chat with pet owner 
- vaccination update
